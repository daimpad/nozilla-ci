# nozilla — Brand & Style Guidelines

All design and verbal guidelines are documented in the **[Corporate Identity Document](./index.html)**.

## Quick Reference

**Colors**
- **Signal-Grün** (#00FF9C) — Buttons, CTAs, markers only
- **Papier-Gelb** (#FFFEE5) — Global background
- **Tinte-Schwarz** (#000) — Text, lines, borders
- Never: gradients, soft shadows, glass effects

**Typography**
- **Display**: Zilla Slab Bold (56–128 px) — Headlines & campaign statements
- **Body**: Inter Regular (17 px) — Body text, line-height 1.55
- **Labels**: Space Mono Bold (11–12 px) — ALL-CAPS, tracking 0.12em

**Form Language**
- **Radius**: Always 0 — sharp corners only
- **Shadows**: Hard offset only (6px 6px 0 0), never blur
- **Stroke widths**: 1.5px (hair), 2px (rule), 3px (strong), 4px (heavy icons)

**Voice & Tone**
- Direct, honest, German language
- Short verbs over long noun chains
- No buzzwords: never "seamless", "disruptive", "synergy"
- Always: "Gute digitale Dienste." (with period)

**Green Marker Element**
- Use `<mark class="g">keyword</mark>` for 1–3 key terms per paragraph
- Semantic only, never decorative
- Never: full sentences, consecutive markers, non-text surfaces

**The 10 Rules**
- ✓ Do: Paper background, ink lines, signal-green for action only
- ✓ Do: Sharp corners, hard shadows, proper type hierarchy
- ✗ Never: Rounded corners, gradients, filters, emoji, stock photos, buzzwords
